# Supply-Chain-KPI-Dashboard
End-to-end inventory and sales analysis using SQL and Excel to track revenue trends, product performance, and regional sales insights.




Interactive Power BI dashboard to monitor supply chain KPIs like on-time delivery, logistics cost, and operational efficiency.


![supply22](https://github.com/user-attachments/assets/434fbe9e-369e-435d-b0fa-9033f604f1d9)


This project presents a Power BI dashboard designed to monitor and analyze key supply chain metrics such as delivery performance, cost efficiency, and order fulfillment.
