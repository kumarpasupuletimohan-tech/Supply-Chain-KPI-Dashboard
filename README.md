# Supply-Chain-KPI-Dashboard


---




This project presents a Power BI dashboard designed to monitor and analyze key supply chain metrics such as delivery performance, cost efficiency, and order fulfillment.

 Objectives
Track on-time delivery performance
 Monitor logistics and transportation costs
 Evaluate supply chain efficiency
 Identify operational bottlenecks



Key KPIs
 On-Time Delivery %
 Average Delivery Delay
 Total Logistics Cost
 Order Fulfillment Rate

 




 Business Impact
Enables better decision-making in logistics, improves delivery performance, and reduces operational costs.

 Future Enhancements
 Integration with real-time data
 Predictive delivery analytics
